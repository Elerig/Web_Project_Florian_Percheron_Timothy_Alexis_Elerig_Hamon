Who are we ?

From the beginning of the realization of this project, we decided to divided up the tasks that we had to do.

Indeed, the fact that this website is composed of several pages allowed us to divide the work while remaining linked.

Each of us think that this project was very interisting due to the fact that we learnt a lot of things about the creation of a website !

We are a team of 3 students. Here, discover our feedbacks and which part was realized by who:

  • Florian
  • Timothy
  • Elerig
  • "For this project, I have been working on the footer. I was surprised how important this was for the user. It gives us information that are very important.

    Futhermore I created the page "Drone's history" and "Our future Drone".
    It was very fun working as a group in a project like this.

    We learnt fundamental knowledge about HTML, CSS and Javascript and I hope we will have other projects like that in the future."
  • "I had no clue about what this webpage was going to cover until our friend Elerig suggested the idea of drones. I was immediatly seduced as I knew little on the subject and wanted to enrich my knowledge on the latter.

    After the distribution of tasks, I decided to handle the "issues" and "domains of application" linked to drones pages and did a pretty good job in finding the right information before organising it into different sections.

    This project enabled me to learn a lot about drones as well as developing my javascript skills which weren't that deep before the creation of this website. I will say teamwork was crucial and helped us get where we wanted, as well as developing another set of human skills.

    Several problems persisted for a little longer than others, like the page layout, the phone version and some javascript effects. With the combined work of the 3 of us, we managed to minimize those issues to make the best out of this project."
  • "During the creation of this website, I was able to create the Home page, as well as the Quiz part but also the page which you are reading actually !

    Being able to learn HTML, CSS and Javascript was something that I found very interesting !
    Making the Home page with effects using Javascript was pretty fun ! Interacting with the user in doing the QUIZ was something new to me, and very nice !

    Even if we decided to divide the realization of this website into different parts, I still could help my collegues for the realization of their own pages.

    Finally, this IT project was not always easy to build but it was really interesting and I consider that we made a good job !"
